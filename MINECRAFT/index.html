Minecraft Notebook

Welcome to my Minecraft Notebook website

In this website there are 2 functionalities:
• First, it's the Notebook function, the Notebook function allows user to write notes about their minecraft journey, this can help keep track of their current progress or things they would like to note down.
• Second, it's the Resource Calculator function, when the user wants to craft a specific amount of items, they can use the resource calculator to calculate how much materials they need to gather in order to craft them.
