About

This website was made to help guide people through my Minecraft playthrough, it features the important functionalities which I need when playing Minecraft, for example, the Notebook function, which proves useful to me as I can do tasks such as taking down notes of locations, to do in the future, what I urgently need, etc...
The Resource Calculator function is also very useful to me as I can calculate how much materials I need to craft an item without going through the hassle of calculating it manualy.

How it's created

HTML

To create the website, HTML is used. This is to create the structure of the website which will then be used to designed using CSS.

CSS

To design the website, CSS is used. The images and colors are all implemented using CSS, as well as the text written.

Javascript

The main goal of this website was to improve my Javascript knowledge, here are the pages that uses javascript to code the functionality of the website.

Local Storage

This page is where the knowledge of localstorage was tested
It allows the user to enter text into the textarea, which will then be saved into the localstorage
So whenever the user wants to look at their notes in the future, the website will get their notes from localstorage and display it to them.
This page has taught me the concept of localstorage and how I can use it in future website, which expanded my understanding of localstorage.

Testing my JS knowledge

In this page is where I used my own knowledge I learnt from online javascript courses to create a resource calculator which will help users calculate how much resources they need to craft an item.
Each item has different crafting formula and different resources, therefore each item will have it's own equation.
This page taught me how to use javascript to manipulate DOM in order to create a more functional website rather than pure HTML and CSS, which allowed me to improve my creativity when making websites.
